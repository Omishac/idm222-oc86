Univsocial

Univsocials Mission:

UnivSocial is a Web application created by three other developers and me in the Computing and Informatics Design II & III class. The main purpose of the app is to create a web application dedicated to Drexel Students where they can find other friends or clubs based on data-matching algorithms that will distribute the best matches for them.

Front-end Developer role:

In this project, my main purpose was to create an aesthetically pleasing interface for the users of UnivSocial. This consisted of creating mood boards of possible logos and color schemes using Canva. After finalizing, I went to Figma to create wireframes of the pages needed to complete the application. To finalize the project, I implemented the color scheme and logo to the wireframes that were made in the Figma application.
